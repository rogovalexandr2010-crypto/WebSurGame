MineGame Advanced Update - автономный Web App
=====================================

Новая версия:
- Алмазы самые редкие, золото очень редкое, железо не очень редкое, уголь часто, камень очень часто
- Встреча с монстрами: выбор "напасть" или "убежать"
- Показывается здоровье игрока и монстра
- Таймер копки, энергия, инвентарь и продажа ресурсов сохраняются

Прогресс сохраняется автоматически через localStorage
