MineGame Advanced - автономный Web App
=====================================

Механика:
- Копка вниз с таймером
- Энергия 20 ед., восстанавливается по 1 за минуту
- Ресурсы начисляются по таймеру случайно
- Шанс встретить монстра (Пещерный медведь, Паук)
- Продажа ресурсов у торговца (кнопка Продать)
- Разные кирки ускоряют копку и увеличивают добычу

Прогресс сохраняется автоматически через localStorage
