MineGame Advanced Update 3 - автономный Web App
=====================================

- Интерфейс возвращён к предыдущему виду
- Выбор действий при встрече с монстром теперь кнопками в Web App
- Боевая система с уроном и расчётом победы
- При смерти игрока выводится сообщение и начинается новая игра
- Алмазы самые редкие, золото очень редкое, железо не очень редкое, уголь часто, камень очень часто
