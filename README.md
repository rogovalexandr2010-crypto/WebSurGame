MineGame Advanced Update 2 - автономный Web App
=====================================

Новая версия:
- Боевая система с уроном
- Победа/поражение зависит от соотношения здоровья и урона
- При смерти выводится сообщение и можно начать сначала
- Алмазы самые редкие, золото очень редкое, железо не очень редкое, уголь часто, камень очень часто

Прогресс сохраняется автоматически через localStorage
